Game controls and options

Game menu

Displayed overlaying the title screen:

  • Story mode - starts with the level 1 cutscene
  • Head-to-head
    • vs CPU - choose character and play against the computer
    • Split screen - 2 players on same system
    • Host network game - create game, set game options
    • Join network game - join an Internet/LAN game by IP
  • Options
    • Game play options - difficulty level, grid overlay, next piece display
    • Video options - 16/32-bit, enabling shadows, blending fx, detail levels, etc.
    • Sound options - sound and music volume, quality (Hz), channels
    • Configure controls - joysticks, keyboard - select 1up/2up device, mappings (actions, bind to button, save, reset to defaults)
  • Credits
  • How to play - instructions, tutorial (no out of game documentation)
  • Quit

Game controls

The following actions can be mapped to any keys/buttons.

  • Move left
  • Move right
  • Rotate piece
  • Accelerate down
  • Launch in-game menu
  • Screen shot
  • Pause
