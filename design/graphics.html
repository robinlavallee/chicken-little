  • Frame overlay graphic, defines playing areas...
  • [UI] Mouse cursors
  • [UI] Menu system font (large, textured)

Week 3 - Sept 11-17, 2000

Week 5 - Sept 25-29, 2000

Post-alpha 1

  • Character artwork (cartoony, animated).
    • Concept sketches
    • Portrait - winning, losing, neutral, tweens
    • Full body - for cutscene, animated talking and movement
  • Game pieces (3d rendered, animated).
    • Textured eggs
    • Treasure chest closed, opened to reveal each bonus (paint brush, horseshoe, gold), opening frames.
  • Special events
    • Fire ball - horizontal from left or right (additive blend)
    • Ice storm - hail with movement trail (additive blend)
    • Quake - "eats" up one column of eggs (overlayed animation)
  • Backdrops. Very colorful, high quality, some animation. Shared between cutscenes and puzzle backdrops.
    • One for each level of story mode (6).
  • UI elements
    • Standard dialogs font ("Connecting to host", etc., re-colorizeable)
  • Story scenes - use backdrops and characters used in the game.
  • Credits screen
  • Web site graphics
