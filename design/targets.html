Targets

Gamedev.net contest

We will complete an alpha version of our game for the Gamedev.net compo. This will help create a schedule for us, and maybe provide us with some cooL prizes.

Only 3 people can be submitted with our entry, and only three prizes will be awarded if we win. Though Kevin seems to be okay with additional people working on the project (I will go without a prize if we win - n8).

August 27, 2000 : Draft design document done.
October 7, 2000 : Alpha 1 and final design document submitted.

The alpha version should not be distributed beyond the gamedev.net site and should have a timeout mechanism (November 15, 2000).

Future versions will be available from gratisgames.com - we will ask that those trying alpha/beta versions submit any bugs they find. Each version will expire - we must release an updated version before the prior expires.

Alpha 2

Complete by: November 1, 2000
Expires: December 15, 2000

  • All in game features implemented - special events, wildcard egg
  • Additive blending, alpha masks, particle system (engine)
  • Drop indicators
  • Animated potraits for existing characters
  • Single file distribution (ZFile)
  • Enhance input device support
  • Head-to-head play mode (same computer)
  • Configuration options

Alpha 3

Complete by: December 1, 2000
Expires: February 15, 2001

  • Game play balancing
  • High score table
  • Improvied AI routines (difficulty levels)
  • All levels of story mode

Beta 1

Complete by: February 1, 2001
Expires: March 31, 2001

This will be the first feature-complete version.

  • Multiplayer (LAN/Internet)
  • How to play (tutorial)
  • Cutscenes

Beta 2

Complete by: March 15, 2001
Expires: April 30, 2001

  • Bug fixes and minor tweaking

Final version

Complete by: April 15, 2001 (Easter Sunday)
Expires: N/A

  • Bug fixes and minor tweaking

Marketing

This is a freeware game, developed by Divine Entertainment and published by Gratis Games. Gratisgames.com will be the official site for the game.

Once the final version is complete, an effort will be made to put our game on popular download sites, possibly in magazines (CDs), and to cross-link with edutainment and egg-related companies.

System requirements

Windows 95/98/ME/NT/2000
Highcolor/Truecolor capable graphics card
Pentium 166 MMX or equivalent (MMX is required)
16-bit sound card highly recommended
